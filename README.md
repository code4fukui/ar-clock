# ar-clock

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

An augmented reality (AR) clock project.

## Features
- Displays the current time as a floating text plane in 3D space
- Uses Three.js and egxr.js libraries for rendering and AR functionality
- Allows PC control using the PCControl class

## Usage
To run the project, follow these steps:

1. Clone the repository:
```
git clone https://github.com/code4fukui/ar-clock.git
```

2. Open the `index.html` file in a web browser.

The clock will be displayed as a floating text plane in the 3D scene. You can use the mouse to interact with the scene and control the camera.
