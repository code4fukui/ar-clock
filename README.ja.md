# ar-clock

拡張現実（AR）時計プロジェクト。

## 機能
- 3D空間に現在時刻を浮遊するテキストプレーンとして表示
- レンダリングおよびAR機能にThree.jsとegxr.jsライブラリを使用
- PCControlクラスを使用したPCからの操作に対応

## 使い方
プロジェクトを実行するには、以下の手順に従ってください:

1. リポジトリをクローンします:
```
git clone https://github.com/code4fukui/ar-clock.git
```

2. ウェブブラウザで`index.html`ファイルを開きます。

時計は3Dシーン内に浮遊するテキストプレーンとして表示されます。マウスを使用してシーンとインタラクトし、カメラを操作することができます。
